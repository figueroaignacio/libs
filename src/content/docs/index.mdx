## React Hooks & TypeScript Utilities

Welcome to the documentation for our collection of React hooks and TypeScript utilities. This library provides battle-tested, type-safe utilities to accelerate your React development.

## What's Included

### React Hooks

Custom React hooks that solve common problems in modern web applications:

- **useLocalStorage** - Persist state in localStorage with automatic synchronization
- **useDebounce** - Debounce values to optimize performance
- **useMediaQuery** - Respond to CSS media queries in your components

### TypeScript Utilities

Type-safe utility functions for common tasks:

- **cn** - Merge Tailwind CSS classes intelligently
- **formatDate** - Format dates consistently across your application

## Getting Started

Browse the documentation using the sidebar navigation. Each hook and utility includes:

- Detailed usage examples
- Full TypeScript definitions
- Implementation source code
- API reference

## Quick Example

```tsx
import { useLocalStorage } from "@your-org/react-utils";

function App() {
  const [name, setName] = useLocalStorage("name", "Guest");

  return (
    <div>
      <input value={name} onChange={(e) => setName(e.target.value)} />
      <p>Hello, {name}!</p>
    </div>
  );
}
```
